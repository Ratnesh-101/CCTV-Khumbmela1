# Venue Security & Incident Playbook (Demo SOP)

## Fight or physical altercation
1. Notify on-site security via radio channel A.
2. Do not broadcast public panic unless medical emergency is visible.
3. Isolate the area; preserve CCTV clips with timestamps.
4. If injuries: call medical and escalate per local law.

## Unattended bag or object
1. Do not touch. Establish 5m perimeter if policy allows.
2. Notify supervisor with camera ID and timestamp.
3. If object remains beyond grace period: follow bomb-threat checklist only if mandated by venue.

## Crowd surge or panic motion
1. Open secondary exits; PA announcement template: calm directional guidance.
2. Security to high-density zones; pause escalators if applicable.

## Acoustic alert (loud transient / possible gunshot-like)
1. Treat as unverified until human confirms on CCTV/audio.
2. Stage alert: security first; public broadcast only after dual confirmation.

## Help request from visitor (mobile/web)
1. Acknowledge receipt; dispatch nearest roving team.
2. Share last known coordinates with accuracy note.

## Weapon-like object (vision, unverified)
1. Never state certainty. Use "possible weapon-like object — verify."
2. Operator must visually confirm before mass notification.

## Aerial / missile-like (vision, custom model — unverified)
1. Consumer CCTV is a poor sensor for real missile warning. **Follow government / military air-raid and emergency apps**, sirens, and official channels first.
2. If a **custom-trained** detector flags `missile` / `rocket` / similar classes, treat as **unverified computer vision** — confirm on multiple feeds or human eyes before any public panic message.
3. Expect heavy false positives (aircraft, drones, birds, lens flare). Log confidence and retain short clips for review.

## Unauthorized drone / UAV (vision, custom model — unverified)
1. **Detection alone does not mean “unauthorized.”** Production systems need **no-fly zones**, **registration/allowlists**, RF/geo cues, and legal process per jurisdiction.
2. Treat CV flags as **unverified**; correlate with flight plans, venue policy, and second camera / operator confirmation.
3. Common false positives: birds, distant aircraft, insects, reflections. Prefer models trained on your camera angles and range.
